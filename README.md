# Calculator App

Калькулятор звичайний - стандартний функціонал, з врахуванням On/Off
On - вмикаємо калькулятор та робимо розрахунки
Off - вимикає калькулятор. При натисканні на кнопки - нічого не відбувається
Математичні розрахунки - через дорівнює, через оператори з продовженням розрахунків
Видалення - видалення першої цифри в матеамтичних розрахунках
